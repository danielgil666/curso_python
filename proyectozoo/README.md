## Descripción
Este programa permite gestionar un inventario de animales clasificados por su especie y características. Utiliza archivos CSV para el almacenamiento persistente de datos y una interfaz moderna en terminal.

## Requisitos
Para ejecutar este programa, necesitas tener instalado:
* **Python 3.8** o superior.
* **Textual**: Librería para la interfaz de usuario.
